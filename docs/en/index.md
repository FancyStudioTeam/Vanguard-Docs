# Welcome to the Vanguard Documentation!

:::warning Documentation Under Development
The documentation **is written by the Vanguard community** and **reviewed by the development team**. If you have any questions, **we recommend checking our support server**.
:::

In this documentation, you'll learn about **all the available features** for **thoroughly customizing Vanguard** and unlocking its full potential.

:::warning Important Notes
**We strongly recommend that you read our [important notes](#important-notes)** before using Vanguard, to **determine whether it can meet your initial needs**.
:::

## A Different Kind of Multifunctional Bot

Unlike other multifunctional Discord bots, Vanguard’s configuration is based on **setting up a YAML file for each different system**.

This type of configuration allows users to **design fully customized systems** to **meet the needs of their Discord community**.

:::warning Prior Knowledge of YAML
We strongly urge users to **have prior knowledge of YAML**. If not, [**we recommend watching the following video**](https://www.youtube.com/watch?v=BEki_rsWu4E).
:::

## Important Notes

Vanguard is designed to help you **build your own systems from scratch** by defining system functionality in **YAML configuration files**.

- If you want a bot that’s easier to configure, **we recommend choosing other alternatives available on [Top.gg](https://top.gg) or in the [Application Directory](https://discord.com/discovery/applications)**.
- If you want to design your systems from scratch, regardless of how long configuration takes, **we recommend choosing Vanguard**.
